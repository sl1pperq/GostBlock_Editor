# Редактор ГОСТ Блок Схем
Простой редактор блок схем в соответствии со стандартом ГОСТ 19.701-90 (ISO 5807-85) от 01.01.1992 с импортном блок схем в формат PDF и собственный формат BGOST. 

## Инструкция по применению
Скачайте архив и распакуйте его в любую папку. Затем, для удобства, создайте ярлык EXE-файла на рабочий стол. Для того, чтобы открыть файл формата BGOST следует использовать функцию "Файл" -> "Открыть" из меню внутри приложения.
